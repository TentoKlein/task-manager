# Task Manager Frontend

## Overview

This is the frontend application for the Task Manager system.
Built using React.js with user authentication and task management features.

---

## Features

- Login Page
- Register Page
- Dashboard Page
- Protected Dashboard
- Create Tasks
- Complete Tasks
- Delete Tasks
- Logout Feature
- User-specific Tasks
- Search Tasks
- Filter Tasks
- Loading State Handling
- Responsive Styled UI

---

## Technologies Used

- React.js
- Axios
- JavaScript
- CSS

---

## Installation

### 1. Navigate to Frontend Folder

cd Frontend

---

### 2. Install Dependencies

npm install

---

### 3. Run Frontend

npm start

---

## Available Pages

| Route      | Description    |
| ---------- | -------------- |
| /          | Login Page     |
| /register  | Register Page  |
| /dashboard | Task Dashboard |

---

## Main Features

### Authentication

Users can:

- Register account
- Login securely
- Logout
- Access protected dashboard

---

### Task Management

Users can:

- Add tasks
- Mark tasks as completed
- Delete tasks
- View only their own tasks
